Ambuj AI ChatBot (Gradio & Hugging Face Spaces)

Prompt-engineered resume chatbot built with Python and Gradio, deployed on Hugging Face Spaces.

Developed a rule-based chatbot using Python and Gradio's ChatInterface, hosted live on Hugging Face Spaces. The bot responds to interview-style questions using pre-defined keyword logic for domains like work experience, certifications, tools, and projects.

**Technologies Used:** Python, Gradio, Hugging Face Spaces.

**Project Scope:** Personal branding, recruiter interaction, and AI-based resume assistant.

**Future Scope:** Model integration via Hugging Face Transformers (e.g., DialoGPT, DistilGPT2). Ability to upgrade with fine-tuned LLMs based on system specifications. Can be scaled to handle semantic queries via NLP libraries (spaCy, transformers).
